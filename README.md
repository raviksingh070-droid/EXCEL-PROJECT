Ecommerce Sales Analysis Dashboard

This project presents an interactive Ecommerce Sales Dashboard built using Microsoft Excel. It provides a comprehensive view of sales performance, profitability, and customer trends through visually appealing and dynamic charts.

🔹 Key Features:
KPI Metrics: Displays total Sales, Profit, Quantity, Number of Orders, and Profit Margin.
Sales & Profit Analysis: Monthly comparison to track performance trends.
Category-wise Insights: Profit distribution across categories like Furniture, Office Supplies, and Technology.
Sales Share Visualization: Donut chart representation of category-wise sales contribution.
Geographical Analysis: State-wise sales distribution using map visualization.
Top Subcategories: Highlights top-performing product segments.
Interactive Filters: Year, Segment, and Region slicers for dynamic data exploration.
🔹 Tools & Techniques Used:
Microsoft Excel
Pivot Tables & Pivot Charts
Slicers for interactivity
Data visualization best practices
🔹 Objective:

To transform raw ecommerce data into meaningful insights that help in decision-making and performance tracking.
